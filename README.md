# Voice Bank Assistant

A voice-controlled banking assistant built using HTML, CSS, and JavaScript.

## Problem Statement
Build a smart assistant that allows users to manage banking operations using voice commands.

## Solution
Voice Bank Assistant enables users to:
- Check balance
- Add income
- Add expense
- View transaction history
- Reset account

It uses the Web Speech API for voice recognition and speech synthesis.

##  Tech Stack
- HTML
- CSS
- JavaScript
- Web Speech API

##  Features
- Real-time balance update
- Voice command processing
- Transaction tracking
- Clean and responsive UI
